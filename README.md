# LINE Rangers Auto Bot (Multi-Instance)

สคริปต์ Python สำหรับทำระบบอัตโนมัติ (Bot) เกม LINE Rangers ผ่าน Android Emulator ตัวสคริปต์ถูกออกแบบมาเพื่อช่วยข้ามโหมดฝึกสอนและสุ่มกาชาหาตัวละครที่ต้องการโดยอัตโนมัติ (Auto Reroll) พร้อมระบบสำรองบัญชีให้ทันทีเมื่อเจอตัวละครที่ตั้งค่าเป้าหมายไว้

## ฟีเจอร์หลัก (Features)

* **รองรับหลายหน้าจอ (Multi-Instance):** จัดการและควบคุม Emulator ได้พร้อมกันสูงสุดถึง 50 จอ
* **ระบบสุ่มกาชาอัตโนมัติ:** เล่นผ่านโหมดฝึกสอนทั้งหมดและกดสุ่มกาชาให้ด้วยตัวเอง
* **ระบบ OCR ตรวจจับตัวละคร:** ใช้ EasyOCR ร่วมกับการทำ Image Processing เพื่ออ่านชื่อตัวละครที่สุ่มได้ และเทียบกับรายชื่อเป้าหมายที่ผู้เล่นตั้งไว้
* **แบ็คอัพบัญชีอัตโนมัติ:** เมื่อสุ่มได้ตัวละครเป้าหมาย ระบบจะดึงไฟล์ข้อมูลบัญชี (`_LINE_COCOS_PREF_KEY.xml`) ออกมาเก็บไว้ในโฟลเดอร์ backup ให้ทันที
* **แผงควบคุมอัจฉริยะ (CLI Control Panel):** แสดงสถานะบอทแต่ละตัว (RUNNING / STOPPED) แบบเรียลไทม์ผ่าน Command Line และสามารถสั่งเปิดปิดบอทรายตัวได้แบบสะดวก
* **ระบบจัดการ Error และ Restart:** มีระบบตรวจจับอาการค้างของ ADB หน้าจอดำ หรือเซิร์ฟเวอร์หลุด และจะทำการรีสตาร์ทตัวเกมหรือการเชื่อมต่อให้ใหม่แบบไร้รอยต่อ
* **แจ้งเตือนผ่าน Discord:** ส่งข้อความแจ้งจำนวนจอที่กำลังทำงานเข้า Discord Webhook อัตโนมัติเมื่อสั่งเริ่มรันสคริปต์

## สิ่งที่ต้องเตรียม (Prerequisites)

* ติดตั้ง **Python 3.8** ขึ้นไป
* ติดตั้งโปรแกรมจำลอง Android Emulator (แนะนำ **MuMu Player 12** สำหรับความสเถียรในการรันหลายหน้าจอ)
* ตัว Emulator ต้องเปิดการตั้งค่า "Root Permission" และ "ADB" ให้เรียบร้อย
* เตรียมไฟล์ `adb.exe` ไว้ในโฟลเดอร์ `bin/adb/` ของตัวโปรเจกต์ให้พร้อม

## การติดตั้ง (Installation)

1. ติดตั้งไลบรารีของ Python ที่จำเป็นทั้งหมดผ่านคำสั่ง:
   ```bash
   pip install pure-python-adb opencv-python numpy rapidfuzz colorama pyperclip requests easyocr
   ```
   *(ข้อแนะนำ: หากใช้งานคอมพิวเตอร์ที่มีการ์ดจอ Nvidia ควรติดตั้ง PyTorch เวอร์ชัน CUDA คู่ไปด้วย เพื่อช่วยให้ EasyOCR ประมวลผลผ่าน GPU ซึ่งจะลดภาระของ CPU ลงได้มหาศาลเวลารันหลายจอพร้อมกัน)*

2. นำรูปภาพเป้าหมายที่ใช้สำหรับการมองเห็นของบอท ไปใส่ไว้ที่ `bin/pic/` ให้ครบถ้วน

## การตั้งค่า (Configuration)

แก้ไขและตั้งค่าระบบได้จากไฟล์ `bin/config.ini`:

```ini
[SETTINGS]
# เลือกหน้าตู้กาชาที่จะให้สุ่ม (เช่น ตู้ที่ 1, 2, หรือ 3)
gachaselect = 1

# จำนวนตัวละครเป้าหมายที่จะสุ่มหา
herowant = 1

# ชื่อหรือคีย์เวิร์ดของตัวละครเป้าหมาย (หากตั้ง 2 ชื่อขึ้นไปให้ใช้ + หรือคั่นด้วยเว้นวรรค)
name1 = ตัวละครตัวอย่าง
name2 = ตัวละครตัวอย่าง2

# จำนวนลูปรอการอ่านภาพสูงสุดก่อนระบบจะเริ่มทำใหม่ (กันภาพค้าง)
max_loop = 40

# สั่งให้บอทตัวใหม่เริ่มทำงานทันทีเมื่อเชื่อมต่อ ADB สำเร็จ
autoStart = True
```

## วิธีใช้งาน (How to Use)

1. เปิดเปิดโปรแกรม Android Emulator จำนวนจอที่ต้องการทั้งหมด
2. เปิด Command Prompt หรือ Terminal ที่ตำแหน่งโฟลเดอร์โปรเจกต์
3. รันคำสั่งเปิดใช้งานแผงควบคุมระบบ:
   ```bash
   python bot.py
   ```
4. ระบบจะแสดงแผงควบคุม (Control Panel) ให้พิมพ์ตัวเลขที่ต้องการเพื่อสั่งหน้าจอทั้งหมด:
   * `[1]` Start All Bots - สั่งเปิดใช้งานบอทบนทุกหน้าจอ
   * `[2]` Stop All Bots - สั่งหยุดบอทบนทุกหน้าจอ
   * `[3]` Individual Bot Control - เลือกสั่งเปิด/ปิดบอทเฉพาะตัวที่ต้องการเจาะจง
   * `[4]` Refresh Connections - ตรวจสอบการเชื่อมต่ออุปกรณ์ ADB ใหม่อีกครั้ง
   * `[0]` Exit Program - ปิดการทำงานและออกจากโปรแกรมอย่างปลอดภัย

## โครงสร้างโปรเจกต์ (Project Structure)

* `bot.py` : เอนจินส่วนกลางและกระบวนการทำงานหลักของโปรแกรม
* `bin/pic/` : โฟลเดอร์เก็บไฟล์ภาพเทมเพลตต่างๆ ที่บอทใช้มองอ้างอิง
* `bin/adb/` : โฟลเดอร์เก็บไฟล์กระบวนการคำสั่งของ Android (ADB)
* `backup/` : โฟลเดอร์สำหรับเก็บไฟล์บัญชีเกมที่สุ่มได้ไอดีเทพแล้ว
